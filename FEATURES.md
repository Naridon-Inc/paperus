# Paperus Features Guide

## New Features (v1.0.1)

### "More" Menu (...)
Located at the top-right of the window, this menu provides access to advanced document actions.

#### Style
- **Font Selection**: Toggle between Default (Sans-serif), Serif, and Mono fonts per document.
- **Small Text**: Reduces font size for denser content.
- **Full Width**: Expands the editor to use the full window width.

#### Actions
- **Copy Link**: Copies a `notionless://` link to the clipboard for sharing.
- **Duplicate**: Creates a copy of the current document in the same folder.
- **Move to**: Open a dialog to move the file to another folder in your workspace.
- **Move to Trash**: Moves the file to the system trash.

#### Tools
- **Lock Page**: Prevents accidental edits. Only the owner or managers can unlock.
- **Import**: Import Markdown files into the current document.
- **Export**: Export the current document as PDF (with layout preserved) or HTML.
- **Notify me**: Toggle subscription to document updates (currently a local preference).

#### Stats
- View detailed Word Count, Character Count, and estimated Reading Time at the bottom of the menu.

## Sharing & Collaboration
- **Share**: Use the "Share File" button in Properties or the "..." menu to open the sharing popover.
- **Invite**: Search for users by name, email, or `@username`.
- **Permissions**: Grant View, Comment, Edit, or Manage access.
- **Links**: Generate public invite links for workspaces.

## Workspace Management
- **Switcher**: Click the workspace name in the sidebar to switch between "Personal" (Local) and Cloud Workspaces.
- **Account**: Manage your profile, set a unique `@username`, and change password in Settings > Account.
